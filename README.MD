# Template of Login

**Login** template of login made in nodejs using API and JSON.

### Dependencies 

- NodeJS
- Express
- npm

### Installing

To run the server, we need to insert the folder that contains the `package.json` file.
1. Installing NodeJS dependencies:

```shell
$ npm install
```

2. Running the server:

```shell
$ npm start
```

#### Login

```

    'email' : <email>,
    'password' : <password>
}
```


result if sucess:

```

```

